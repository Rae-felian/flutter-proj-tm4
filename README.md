## Getting Started

1. Clone repository ini
2. Jalankan `flutter pub get`
3. Run dengan `flutter run`

- Flutter
- Riverpod (state management)
- Material 3

## Form Fields

- Nama Lengkap (required)
- NIM (required, 8+ digit)
- Email (required, valid format)
- Nomor Telepon (required, 10+ digit)  
- Alamat (required)
- Pesan (optional)
